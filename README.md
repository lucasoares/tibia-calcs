# tibia-calcs

Useful calculators and statistics for Tibia MMO.

## Structure

This project has two main directories:
- [frontend](frontend)
- [backend](backend)

Our frontend uses [Vue JS Framework](https://vuejs.org/).

Our backend uses [Spring Framework](https://spring.io/) with Java 14.
